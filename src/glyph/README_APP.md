# Glyph Annotation Tool

Lightweight annotation tool for image detection built for Google Cloud Workbench.

## Quick Start in Workbench

1. Create a new Custom App in Vertex AI Workbench
2. Specify:
   - Repository: `https://github.com/verily-src/workbench-app-devcontainers.git`
   - Branch: `master`
   - Path: `src/glyph`
3. Wait for deployment (~2-3 minutes)
4. Access the app URL and start annotating!

## Features

- Bounding box annotation with interactive canvas
- Auto-save to BigQuery or local storage
- Export to COCO format for ML training
- Multi-user support
- 97% cost savings vs Label Studio
